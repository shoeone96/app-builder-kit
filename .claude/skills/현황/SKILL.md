# /현황 — 진행 상황 확인

지금까지의 진행 상황을 보여주는 스킬.

## 트리거

"현황", "어디까지 했어?", "상태", "뭐가 완성됐어?"

## 실행 흐름

### Step 1: 문서 읽기

`docs/progress.md` 읽기.
없으면: "아직 시작한 기능이 없어요. `/시작`으로 프로젝트를 만들어볼까요?"

### Step 2: 3가지 분류로 표시

```
📊 지금까지 진행 상황

✅ 완성된 기능
- 회원가입 (2월 15일 완성)
- 로그인 (2월 16일 완성)

🔨 만드는 중
- 게시글 작성 (서버 완성, 화면 작업 중)

📋 기획만 된 기능 (아직 안 만듦)
- 채팅 (기획 완료)
- 알림 (기획 전)
```

### Step 3: 추천

"다음에 뭘 하면 좋을까요?"
- 만드는 중인 것 → "이거 마저 만들까요?"
- 기획만 된 것 → "이 기능을 만들어볼까요?"
- 기획 전인 것 → "이 기능을 기획해볼까요?"
